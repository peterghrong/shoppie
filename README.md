# Shopify summer 2021 internship coding challenge

## How To Use
Clone the repository
```python
cd myproject
pip -r install requirements.txt
```
and use 
```python
flask run 
``` 
to host the webapp locally at
```http://127.0.0.1:5000/```

## What I have implemented
* Photo Upload
* Multiple Photos Upload
* Sqlite3 database User signup
